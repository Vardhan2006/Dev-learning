Hooks:-

1. useSate:- To manage the states
2. useEffect:- It manages the side effects in a functional components like API Calls, DOM manipulation, event listener
3. useRef:- To hold the mutable values so that it does not trigger render
4. useContext:- to manage the global state.
5. useMemo and useCallback:- For optimization, to avoid unnecessary re-renders.
6. useReducer: To manage complex state management like mini version of redux
